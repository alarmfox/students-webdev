# Esercitazioni di Sviluppo Web

Questa repository contiene una serie di esempi ed esercizi pratici utilizzati durante il corso di sviluppo web. Ogni cartella è pensata per illustrare un concetto specifico attraverso codice commentato e task pratici.

## Come iniziare

### Requisiti
- Un browser moderno (Chrome, Firefox, Edge, Safari).
- Un editor di codice (si consiglia [Visual Studio Code](https://code.visualstudio.com/)).
- [Git](https://git-scm.com/) installato sul sistema (opzionale ma consigliato).

### Clonare la repository
Per scaricare i file sul proprio computer, apri il terminale e digita:

```bash
git clone https://github.com/alarmfox/students-webdev.git
```

## Struttura della Repository

Gli esercizi sono organizzati per lezione e argomento:

### Lezione 1: Fondamenta del Web & DOM
- **1.base**: Introduzione al Document Object Model (DOM) e agli oggetti globali `window` e `document`.
- **2.children**: Navigazione dell'albero DOM e manipolazione dei nodi figli.
- **3.events**: Gestione dell'interattività tramite i listener di eventi.
- **3.forms**: Gestione base dei moduli (form) e recupero dati.
- **4.forms-modular**: Esempio di organizzazione modulare con file `.html`, `.js` e `.css` separati.
- **5.lists**: Creazione e manipolazione dinamica di liste di elementi.
- **6.frameworks**: Confronto tra l'utilizzo di framework classici (Bootstrap) e approcci moderni.
- **7.async**: Introduzione alla programmazione asincrona e alle chiamate API.

## Come utilizzare gli esempi
1. Entra nella cartella dell'esercizio che ti interessa.
2. Apri il file `index.html` trascinandolo nel browser o facendo tasto destro -> "Apri con".
3. Apri la **Console degli sviluppatori** (F12 o Tasto destro -> Ispeziona -> Console) per visualizzare i messaggi di log e gli errori.
