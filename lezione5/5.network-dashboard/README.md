# Esercizio 5: Network Inventory Dashboard

Un caso d'uso reale nel mondo del networking: una dashboard per monitorare apparati di rete (Router).

## Obiettivi Didattici
- Gestione di strutture dati complesse e annidate.
- Visualizzazione di liste dentro liste (Router -> Interfaces).
- UX avanzata: stati di caricamento, gestione errori e feedback visuale.
- Layout professionale a griglia interamente realizzato con **Tailwind CSS**.

## Requisiti
È necessario il backend della Lezione 4 attivo:
```bash
cd ../../../lezione4/network-api/server
npm install
node app.js
```

## Come avviarlo
1. `npm install`
2. `npm run dev`
